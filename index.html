API Resources

The A-List of APIs.

Foursquare

Search for Places in an Area, Connect With Foursquare Users, Know When Somebody Checks In, Get Global Streaming Check-In Data

 Foursquare

Eventbrite

Used for event organizing and awareness. Really, it's a lot cooler than I can make it sound.

 Eventbrite

Yelp

Search for businesses by location (not just for resturants/food, keep that in mind!), keyword and category. Query Yelp attributes such as photo, rating, menus and review snippets.

 Yelp

Zillow

Get Real Estate data on Apartments and Housing nationwide.
Search by Zip or Address, and it will geocode those addresses for you!

 Zillow

Facebook

Should be self explanitory. Get Facebook posts, events and user data.

 Facebook

Reddit

Get reddit data from any subreddits! AND UNLIMITED* PHOTOS OF CATS!!!

* Not Actually Unlimited.
 Reddit

WalkScore

Neighborhood ratings for a ton of apartment complexes/houses nationwide. They use a "complex" algorithm to score areas.

 WalkScore

Google API

The best map building API on the market today. Plus, you can make it look like an old school pirate map. It's pretty much baller status. Is that still a thing people say?

Pintrest

 Pintrest

Twilio

Send SMS messages to phones from your app. It does some other stuff too. Don't ask me why, just use it (if you want to).

 Twilio

Mashape

 Mashape

Wireframe Resources

Visualize your web page.

Draw.io

Free service that is mainly used for flow charts, but works well for wireframing. If you are familiar with Google services, this would be a good resource to use for wireframing. Not a whole lot of features outside of building mockups, but definitly servicable.

 Draw.io

Sketch

Sketch is $99, however they have a 30 day free trial. I would highly reccommend trying it out. I personally use sketch for wireframing and logo design. One of the more notable features is CSS exporting!

 Sketch

NinjaMock

NinjaMock is a free service specifically for building website wireframes. A bit more complex than the previous, it boasts a wealth of features: creating and edit vector graphics, editable SVG icons, can be setup for click through and usability testing.

 NinjaMock

InVision

InVision is a free wireframe specific service. Takes a bit of setup, but it allows you to create and navigate through your wireframes and build near accurate click throughs.

 InVision

Balsamiq

A very simple wireframing service. It's $89 for the single user version, but they do have a free trial. It is dead simple to use and you can quickly create nice mockups using it. I used it before purchasing Sketch, often finishing my mockups just before presentation.

 Balsamiq

Zeplin.io

Another website specific wireframe builder. It has a free tier and has very robust features. One cool feature is color paletting, allowing you to reference the same palette across multiple pages. Also features Sketch importing.

 Zeplin.io
