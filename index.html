My Favorite APIs

The A-List of API's

Foursquare

Search for Places in an Area, Connect With Foursquare Users, Know When Somebody Checks In, Get Global Streaming Check-In Data

 Foursquare

Eventbrite

Used for event organizing and awareness. Really, it's a lot cooler than I can make it sound.

 Eventbrite

Yelp

Search for businesses by location (not just for resturants/food, keep that in mind!), keyword and category.
Query Yelp attributes such as photo, rating, menus and review snippets.

 Yelp

Zillow

Get Real Estate data on Apartments and Housing nationwide.
Search by Zip or Address, and it will geocode those addresses for you!

 Zillow

Facebook

Should be self explanitory. Get Facebook posts, events and user data.

 Facebook

Reddit

Get reddit data from any subreddits! AND UNLIMITED* PHOTOS OF CATS!!!

* Not Actually Unlimited.
 Reddit

WalkScore

Neighborhood ratings for a ton of apartment complexes/houses nationwide. They use a "complex" algorithm to score areas.

 WalkScore

IBM Watson

Too many options to post all of them. Get personality profiles, sentiment analysis, keyword recognition, and much, much more.

 Watson

Google API

Again, so many options here to use, I would break the internet listing them all.
Get access almost all of Googles Services including: Maps, Trends, Ads, Cloud Monitoring, Calender, etc...

 Google

Mapbox

The best map building API on the market today. Plus, you can make it look like an old school pirate map. It's pretty much baller status. Is that still a thing people say?

 Mapbox

Pintrest

Pintrest. I don't know much about it. This is the API for it. They get a lot of traffic and data so I put it here. I think its a site for making shelves out of whatever you have around the house.

 Pintrest

Twilio

Send SMS messages to phones from your app. It does some other stuff too. Don't ask me why, just use it (if you want to).

 Twilio

Mashape

A search engine for APIs. You'll find some pretty strange stuff here. Probably a good number of broken and/or useless APIs.

 Mashape
